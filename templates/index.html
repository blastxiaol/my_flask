Projects

Prediction Model based on Bayesian Hierarchical Clustering
Bayesian Analysis, Summer 2019. [Poster]

We applied Bayesian Hierarchical Clustering algorithm to make roughly correct prediction with limited data provided. We processed, re-generated and achieved efficient algorithms in Julia.

Design of Evaluative System for Online Car Hiring Service
Capstone Porject, Fall 2018. [Code]

We design a new evaluation system with UI which can capture opinions from customers' comments based on NLP and aspect level sentiment analysis by applying China-Word-Vector & Recurrent Attention Network.

Driver's Alarming System in Vehicle
VG100 Porject, Summer 2016.

We built a device which can send alarm to driver when he is sleepy by capturing picture and then detecting whether the driver's eyes are closed or not. The device was assembled in a Raspberry Pi by running Python code with OpenCV.
